Support

Need help with the app, want to report a bad trail marker, or spotted a name that isn't right? You've got a few ways to reach me.

Email anthony.compofelice@centricfiber.com

I read every message and reply within a few days. If it's a data issue (closed bridge, flooded pathway, wrong name), the fastest way is to use the built-in Report a problem flow — About tab → Feedback → Report a problem. It pre-fills your location so I can pinpoint the spot.

Common questions

The map isn't showing my location. Check Settings → Privacy & Security → Location Services → Woodlands Trail Guide is set to "While Using the App." The app never uses location in the background.

A trail is missing / wrong name / closed. Trail data comes from The Woodlands Township GIS. If they've updated it, the app refreshes on the next launch. If it looks wrong to you too, please report it via the in-app form or email me — I'll flag it with the Township if it's their side, or push a fix if it's ours.

The weather is out of date. Tap the weather pill in the top-left of the map, then the refresh arrow in the top-left of the sheet that opens.

Routing sent me an odd way. Try tapping "Add stop" on the summary card and dropping a waypoint on the trail you want to walk — the app will reroute through it. If you think the routing is genuinely broken for a specific area, please email me the two points you tapped.

How do I get rid of the ads? The banner ad supports development. There's no ad-removal option today, but a future update will add one.

Data and privacy

All trail data comes from The Woodlands Township's public GIS. Weather is from Open-Meteo. The app has no accounts, no analytics, and nothing you do leaves your phone unless you tap Report a problem or the ad-tracking prompt.
